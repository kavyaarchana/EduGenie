Project Name: EduGenie

Project Planning:

Objective:
To develop an AI-powered educational assistant that helps students learn through AI-based explanations, question answering, quiz generation, summarization, and personalized learning paths.

Phase 2: Requirement Analysis

Phase 3: System Design
Duration: Week 2
Activities:

Phase 4: Development
